# Table.delete()

​​Table.delete()​ 和 ​​Table.add()​ 接受的值一样

  • 可以是 Table.delete(id)，也可以是 Table.delete({ key: 'value' })
  • 当只有定义了 Schema，才能使用 id 以外的字段


import GoDB from 'godb';

const testDB = new GoDB('testDB'); // 连接数据库
const user = testDB.table('user'); // 获取数据表

doDelete();

async function doDelete() {
   
 const luke = await user.add({
   name: 'luke',
   age: 22
 });
 
 await user.delete(luke.id);
 // await user.delete({ id: luke.id }); // 与上面等价
}
