# Table 操作

在连接好数据库后，增、删、改、查 操作都是以表（Table）为单位操作的

import GoDB from 'godb';

const testDB = new GoDB('testDB'); // 连接名为 testDB 的数据库

const user = testDB.table('user'); // 连接 user 表
const message = testDB.table('message'); // 连接 message 表


然后，你就可以在 ​​user​ 或 ​​message​ 上面进行操作了


GoDB 提供了几个简单的方法，用于增删改查：


  • Table.add()：增加数据
  • Table.get()：查找数据（使用索引）
  • Table.put()：更新数据
  • Table.delete()：删除数据
  • Table.addMany()：一次增加多个数据
  • Table.find()：查找数据（使用查找函数）


灵活运用这几个方法，你就能很好的完成自己的业务逻辑
