# Hello World

首先需要安装，这里默认你使用了 webpack、gulp 等打包工具，或在 vue、react 等项目中


npm install godb


在第一个正式版发布后，还会提供 CDN 的引入方式，敬请期待~



操作非常简单，增、删、改、查各只需要一行代码：


import GoDB from 'godb';

const testDB = new GoDB('testDB');
const user = testDB.table('user');

const data = {
 name: 'luke',
 age: 22
};

user.add(data) // 增
 .then(luke =&gt; user.get(luke.id)) // 查，等价于 user.get({ id: luke.id })
 .then(luke =&gt; user.put({ ...luke, age: 23 })) // 改
 .then(luke =&gt; user.delete(luke.id)); // 删
